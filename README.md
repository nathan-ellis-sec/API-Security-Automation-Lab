# API Security Automation Lab

This repository contains hands-on examples of using real security APIs to automate enrichment, investigation, and Zero Trust workflows. These projects demonstrate my ability to authenticate to APIs, send requests, parse JSON responses, and build practical automation tools.

## What's Inside
- **VirusTotal URL Lookup (Easy)** — Basic GET request, API key usage, JSON parsing.
- **IPinfo IP Enrichment (Medium)** — ASN, geolocation, privacy flags, structured output.
- **Zscaler API (Hard)** — Token-based authentication and real identity or Zero Trust workflows.
